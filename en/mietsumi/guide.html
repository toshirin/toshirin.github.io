User Guide

Set the number of blocks at each grid position, then move the 3D view to examine the model.

1. Build with blocks

Tap the cells under “Height (bottom is front)” to set the number of blocks at each position.

Compare the grid with the blocks in the 3D view as you build. The total number of blocks appears below the app name.

2. Choose a size

Choose the available building area from these sizes:

  • 3×3×3
  • 5×5×5
  • 7×7×7

The available sizes are planned to differ between the free and Full Versions.

3. Rotate and zoom

Drag the 3D view to rotate the model freely, and pinch to zoom in or out.

Use these gestures to inspect the back of a shape or areas where blocks overlap.

4. Change the viewing direction

Under “View,” you can switch directly to:

  • Angled View
  • Front View
  • Left View
  • Right View
  • Back View
  • Top View

5. Change the display mode

Under “Display,” choose Normal, Transparent View, or Wireframe View.

Normal

Shows the faces of each block normally.

Transparent View

Makes the blocks translucent so that obscured areas are easier to inspect.

Wireframe View

Focuses on block outlines. Use it to examine the boundaries of a shape and how blocks overlap.

6. Add color

Choose a color under “Color,” then tap a block in the 3D view to paint it.

Coloring blocks by position or height can make the structure easier to organize and understand.

7. Show the count

Turn on “Show Count” to display the total number of blocks below the app name.

See Learning with MieTsumi for a detailed explanation of Antenna Markers.

8. Reset colors or positions

The “Reset Color” button returns every block to white without changing the arrangement.

“Reset View” returns the camera to the angled position. When Light and Shadow is on, the button becomes “Reset Light” and returns the light to its initial position.

9. Clear all

“Clear All” removes every placed block and its color.

If blocks are present, a confirmation dialog appears before they are removed.

10. Save to Photos

Tap “Save to Photos” to save an image of the current block model to the device’s photo library.

If the app requests permission to add photos, allow it to continue. Saving occurs only when you tap the button, and the image is not sent to the developer.
