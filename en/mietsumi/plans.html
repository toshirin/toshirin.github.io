Plans & Support

Free and Full Versions

Features currently planned for release.

Currently planned features

The following structure is currently planned and may change before the App Store release.

Free Version

  • 3×3×3 building area
  • Basic colors
  • Normal display
  • Transparent View
  • Wireframe View
  • Free rotation and zoom
  • Angled, Front, Left, Right, Back, and Top Views
  • Total block count
  • Antenna Markers

The Free Version is planned to support basic block problems and counting with Antenna Markers.

Planned Full Version additions

  • 5×5×5 building area
  • 7×7×7 building area
  • Additional colors
  • Light and Shadow
  • Save to Photos

The Full Version is planned as a one-time in-app purchase, not a subscription. Pricing will be decided before the App Store release.

Restore purchases

After purchasing the Full Version, users are expected to be able to restore it on devices using the same Apple Account.

Purchase and restoration features are currently in preparation.
